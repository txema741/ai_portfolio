# Portafolio de IA aplicada (Consultoría y Docencia)

**Qué es**  
Portafolio con ejemplos reales de IA usando **ChatGPT Plus** y herramientas **gratuitas**. El objetivo es entregar resultados listos para **decidir** en empresa o **usar** en el aula.

## Qué hago
- **Educación/Docencia**: packs de clase (plan, rúbrica, quiz).
- **Consultoría IA**: diagnóstico de madurez + informe de 1 página.
- **Comercio exterior**: radar de señales (riesgo/oportunidad) y brief.
- **Geopolítica**: extracción de señales y escenarios (O/B/P).

## Cómo trabajo (resumen)
1. Recibo contexto y fuentes.
2. **Agente** (Router → Doer → Crítico) genera y valida la salida.
3. Entrego en **Docs/Sheets/Email** con revisión humana.
