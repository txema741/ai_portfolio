# Acuerdo de Nivel de Servicio (SLA)

- **Tiempo de respuesta**: generación ≤ 2 minutos + revisión ≤ 10 minutos.  
- **Disponibilidad**: días laborables (CET).  
- **Incidencias**: respuesta ≤ 24h.  
- **Calidad mínima**: QA ≥ 80/100 (ver `qa_rubric.md`).  
- **Fuentes y fechas**: obligatorias, si faltan → **N/D**.  
- **Checklist ético**: obligatorio antes de entregar.
